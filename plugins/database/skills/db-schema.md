# Database Schema

Explore and document database schema structure.

## Usage

When asked about database schema:

1. Connect to the target database
2. Query information_schema or equivalent
3. Present tables, columns, types, and relationships

## Features

- List all tables and views
- Describe table columns with types and constraints
- Show indexes and foreign key relationships
- Generate ER diagrams in text format
