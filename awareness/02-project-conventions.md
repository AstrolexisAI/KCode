When creating new projects in this workspace:
- All projects go under the current working directory
- Use descriptive directory names in kebab-case (e.g., bakery-website, glucose-tracker)
- Always verify the project works end-to-end before reporting completion
- When creating web servers, use a proper static file server that handles MIME types correctly
- Never use `python3 -m http.server` for anything beyond quick testing — use Node.js http, express, or similar for production-like serving
